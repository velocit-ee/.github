# code of conduct

## the short version

be useful, be honest, don't be a jerk. help people debug things. share what you know.

---

## the longer version

velocit.ee is a technical project. the people here are trying to build infrastructure tooling that actually works. the expectation is:

**do:**
- help others debug problems, even if the question seems basic
- share your environment details when reporting issues — hardware matters here
- disagree on technical grounds with evidence
- document what you tried before asking for help

**don't:**
- dismiss people for not knowing things
- gatekeep based on background, age, institution, or job title
- post off-topic content, spam, or self-promotion unrelated to the project
- use the project spaces to harass anyone

---

## enforcement

issues that violate this code of conduct will be handled by removing the content and, for repeated or serious violations, restricting access. there is no appeal process for obvious bad behavior.

for reports, email [inquiries@velocit.ee](mailto:inquiries@velocit.ee).

---

this code of conduct is intentionally brief. it is not a legal document. use judgment.
